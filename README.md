# AI Chatbot

This is a Java-based console chatbot developed as part of the CodeAlpha Java Internship (Task 3).

The chatbot interacts with users using rule-based logic and responds to frequently asked questions.

## Features
- Interactive chatbot
- Rule-based responses
- Menu-free conversational interface
- Uses OOP concepts

## Technologies Used
- Java
- VS Code
- Scanner
- Object-Oriented Programming
